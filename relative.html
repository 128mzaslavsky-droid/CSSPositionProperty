Understanding relative paths in HTML

A relative path in HTML specifies the location of a resource relative to the current document. This means that the path is based on the directory structure of your website or project.

Example of a relative path:

If you have an image located in a folder called images within the same directory as your HTML file, you would use the following HTML code to include it: 

<img src="images/photo.jpg" alt="Example Photo">

Key points to remember about relative paths:

  • Relative paths are based on the location of the current HTML file.
  • They are useful for linking to resources within the same website or project.
  • Using relative paths makes your code more portable and easier to move between different environments.
  • Be mindful of the directory structure to ensure the paths are correct.
